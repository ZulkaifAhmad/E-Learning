## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Ensure you have the following installed:
* [Node.js](https://nodejs.org/) (v14.0.0 or higher)
* npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/ZulkaifAhmad/E-Learning.git](https://github.com/ZulkaifAhmad/E-Learning.git)
    cd E-Learning
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    # or if using Create React App:
    npm start
    ```

4.  **Open the app**
    Open [http://localhost:5173](http://localhost:5173) (for Vite) or [http://localhost:3000](http://localhost:3000) (for CRA) in your browser.
